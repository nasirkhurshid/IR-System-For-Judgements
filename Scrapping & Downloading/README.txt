1. "dataScrapper.py" is used to get details of all the cases from Supreme Court's website.
2. "description.txt" (required) contains details about all the cases scrapped through (1).
3. "cases_count.txt" (required) contains details about number of cases lying in each category.
4. "links.txt" contains download links of all the documents/cases scrapped through (1).
5. "pdfDownloader.py" is used to download all the PDFs from the links given in (4) and categorize them in folders.
